----------- Exercise 2 -----------

Multiplication table for numbers up to 12.

Problem Statement

Write a program that prints a multiplication table for numbers up to 12.

Plans and pseudocode

Ask user to input a value in order see its multiplication table up to 12
Take user input value, multiply by one, and add the user input
Use for loop in order to make sure the value is added no more than eleven times in order to keep the table from going beyond 12
i++ is used to increment the values by 1

Demonstration

Input a value into the pop-up window at the top of the page in order to view its multiplication table up to 12.
Refresh the page to input another value.

----------- Exercise 3 -----------

Calculate Total Cost with Variable Tax and Tip

Problem Statement

Write a program that asks the user for a meal price and then calculates the total bill including rates for tax and tip.

Plans and pseudocode

Ask user to input value of the bill, tax percentage, and tip percentage.
Multiply value of the bill by value of tax/100 and add it to the bill value to get the cost with tax included
Multiply value of the bill by the value of the tip percentage/100 and add it to the cost with tax included in order to calculate total cost
Display total cost

Demonstration
